What it does

AutoFill Pro is a QA-engineer's autofill tool for repetitive web forms — capture once, replay anywhere. Built specifically for multi-step wizards (Angular, React, Vue) where the same test data needs to flow across many pages.

Features

Visual pickerClick any field to save its selector and value
One-keystroke replayAlt+F fills the page, in capture order
SPA-awareAuto-fires on the next wizard step
Smart selectorsPrefers id, data-testid, name over fragile paths
Custom dropdownsreact-select, downshift, headlessui supported

Shortcuts

  • Alt+P — start the visual picker
  • Alt+F — auto-fill the page
  • Ctrl/Cmd+Z — undo last capture
  • Esc — stop the picker

Privacy

Zero data collection. Everything stays in chrome.storage.local on your machine. No analytics, no servers, no network calls. Read the privacy policy.
